# 技术规格

## Remotion 版本策略
- 基于 Remotion 官方发行版开发（如 v4.0.469）
- 不使用开发版/主分支，确保稳定性
- 版本锁定，避免自动更新导致兼容性问题
- 后续升级需经过充分测试

## 技术选型
- **GUI 框架**：Electron
- **前端 UI**：React + shadcn/ui + Tailwind CSS
- **动画引擎**：Remotion（官方发行版）
- **后端服务**：Node.js
- **数据分析模块**：Python（用于处理数据动画相关的数据解析和转换）
- **Agent 框架**：LangChain（用于构建多模态对话 Agent）

## LLM 与 Agent
- 基于 LangChain 构建对话 Agent
- 支持多模态模型（图片参考 + 文字描述）
- LLM 提供商可配置（OpenAI、Claude、本地模型等）
- Agent 负责：
  - 解析用户自然语言需求
  - 解析参考图片的视觉结构
  - 生成 Remotion 组件配置
  - 执行对话调整指令

## 通信方案
- **Python 与 Node.js 通信**：采用 Python FastAPI 本地 HTTP 服务
- Node.js 作为主进程管理 Python 子进程生命周期
- 通过 HTTP 请求调用 Python 数据分析接口

## 部署模式
- 纯本地运行，无需用户系统和云端渲染
- 利用本地算力，保护用户隐私
