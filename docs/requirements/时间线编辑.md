# 时间线编辑

## 多轨道编辑功能（完整时间线编辑器）
- **可视化时间线**：支持缩放、滚动、播放头控制
- **多轨道管理**：支持视频、音频、图片、文字、动画、数据图表等多种轨道类型
- **拖拽编辑**：支持素材拖拽调整位置、时长、裁剪
- **关键帧系统**：支持属性关键帧设置（位置、缩放、旋转、透明度等），Remotion 原生支持
- **图层控制**：图层叠加、混合模式、锁定、隐藏、排序
- **轨道操作**：添加/删除轨道、轨道分组、轨道独奏/静音
- **实时预览**：时间线操作与 Remotion 预览联动
- **AI 辅助**：通过对话自动添加/调整轨道和关键帧
