# UI 布局设计

## 整体布局
采用经典剪辑软件三栏+底部时间线布局，但强化对话交互区域。

```
┌──────────────────────────────────────────────────────────────┐
│                        顶部菜单栏                              │
│  [文件] [编辑] [视图] [帮助]                    [渲染] [导出]   │
├────────────┬──────────────────────────┬──────────────────────┤
│            │                          │                      │
│  左侧面板   │        中间工作区         │      右侧面板         │
│            │                          │                      │
│  项目/素材  │   ┌──────────────────┐   │   属性面板            │
│  库        │   │                  │   │   (选中元素时显示)     │
│            │   │   预览窗口        │   │                      │
│            │   │   (Remotion)     │   │   - 位置/尺寸         │
│            │   │                  │   │   - 动画参数          │
│            │   └──────────────────┘   │   - 关键帧            │
│            │                          │   - 样式              │
│            │   ┌──────────────────┐   │                      │
│            │   │   对话面板        │   │──────────────────────│
│            │   │   (可收起)        │   │                      │
│            │   │                  │   │   素材库              │
│            │   └──────────────────┘   │   (拖拽到时间线)       │
│            │                          │                      │
├────────────┴──────────────────────────┴──────────────────────┤
│                        底部时间线                              │
│  [轨道1] ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━       │
│  [轨道2] ━━◆━━━━━━━━◆━━━━━━━━━━━━━━━━━━━━━━━━━━━━━       │
│  [轨道3] ━━━━━━━━━━━━━━━━━━━━◆━━━━━━━━━━━━━━━━━━━       │
│  ◀──▶ [播放头]                                          [缩放] │
└──────────────────────────────────────────────────────────────┘
```

## 各区域功能

### 顶部菜单栏
- **文件**：新建项目、打开项目、保存、导入素材
- **编辑**：撤销、重做、复制、粘贴、删除
- **视图**：面板显示/隐藏、全屏预览、布局切换
- **帮助**：文档、快捷键、关于
- **渲染**：渲染当前子项目
- **导出**：导出视频/工程文件

### 左侧面板（项目/素材库）
- **项目树**：显示总项目 → 子项目层级结构
- **素材库**：
  - 已导入素材列表（图片、视频、音频、数据文件）
  - 拖拽素材到时间线或预览窗口
  - 支持搜索和筛选
- **预设库**：Remotion 官方预设 + 用户自定义预设

### 中间工作区
#### 预览窗口（上方）
- Remotion 播放器
- 播放/暂停/逐帧控制
- 显示当前帧时间码
- 支持全屏预览
- 点击元素可选中并在右侧面板编辑

#### 对话面板（下方，可收起）
- 聊天式交互界面
- 支持文字输入 + 图片上传（参考排版）
- 显示 AI 回复和操作历史
- 可展开/收起，不遮挡预览

### 右侧面板（属性面板 + 素材库）
#### 属性面板（选中元素时显示）
- **变换**：位置、缩放、旋转、透明度
- **动画**：入场/出场动画、缓动函数、持续时间
- **样式**：颜色、字体、边框、阴影
- **关键帧**：关键帧列表，支持添加/删除/编辑
- **数据绑定**：数据动画的参数配置

#### 素材库（未选中元素时显示）
- 与左侧素材库功能类似，方便拖拽操作

### 底部时间线
- **轨道列表**：视频、音频、文字、数据图表等轨道
- **播放头**：可拖拽，显示当前帧位置
- **缩放控制**：时间线缩放级别
- **轨道操作**：
  - 添加/删除轨道
  - 轨道锁定/隐藏/独奏/静音
  - 拖拽调整素材位置和时长
  - 关键帧标记显示
- **播放控制**：播放/暂停、上一帧/下一帧、循环播放

## 交互细节
- 面板支持拖拽调整宽度
- 面板可收起/展开
- 支持自定义布局保存
- 快捷键提示悬浮显示
