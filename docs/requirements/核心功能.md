# 核心功能

## 多模态对话式需求输入
- 支持自然语言描述 + 图片参考（截图、排版参考图等）
- 多模态 LLM 解析视觉布局（元素位置、层级、风格、配色）
- 支持上传视频素材、音频、文案、数据文件等
- LLM 理解需求并转化为 Remotion 可执行的参数

## 核心工作流：参考文稿 + 参考排版 → 动画生成
1. 用户输入参考文稿（文案/脚本/数据）
2. 用户上传参考动画排版图片
3. Agent 解析并自动生成 Remotion 动画
4. 实时视频预览
5. 对话微调，满意后导出

## 动画生成能力
### 重点动画类型
- **数据动画**：图表、数据可视化、数字滚动、进度条等
- **图标动画**：图标入场、变形、路径动画等
- **文字动画**：标题动画、字幕效果、打字机等

### 其他动画类型
- 转场效果
- 粒子效果
- 形状动画
- 相机运动

### 官方预设库
- 集成 Remotion 官方预设
- 用户可直接选择预设并自定义参数

## 输出格式
- **视频文件**：直接渲染输出 MP4 等格式
- **可编辑工程**：导出 Remotion 项目代码，供有能力的用户二次修改
- **可选打开 Remotion 原生 Web 端**：有编程能力的用户可直接进行代码级修改
