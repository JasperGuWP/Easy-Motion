## 1. 目标

完善素材全生命周期：导入（含桌面拖入）、元数据、删除、库 UI、预设；为 Remotion `staticFile` 与 M5 Agent `importAsset` 提供**唯一可信**的素材管道。

## 2. 范围

### In scope

- 桌面拖入：素材库、**时间线**、预览区（见子任务 M6-01/02）
- 格式校验（扩展名 + 魔数）、批量与进度
- 复制到 `项目/assets/` 与 `remotion/public/assets/`（与 `publicPath` 一致）
- 删除素材（软删除 + 时间线引用检查）
- 长素材时长与成片自动延长（M6-05）
- 素材库 UI：分组、搜索、元信息、悬停预览
- SQLite 素材表（M6-08）或 manifest 双写过渡期
- 预设浏览、应用、保存

### Out of scope（属 M5 / M7）

- 对话面板、LLM、Agent 工具编排 → **M5**
- 关键帧编辑、Ripple 删除 → **M7**
- 项目级「成片时长」设置页（可 M7 或项目设置补强）

### M4 已完成（勿重复开发）

- 素材库内拖到时间线、Delete 删片段/轨道、撤销栈

## 4. 开发规范

### 4.1 路径约定（强制）

| 概念 | 路径 |
|------|------|
| 项目内素材 | `{projectRoot}/assets/{type}/{uuid}.ext` |
| Remotion 静态文件 | `{remotion}/public/assets/{type}/{uuid}.ext` |
| `publicPath` | `/assets/{type}/{uuid}.ext` |
| Generator 引用 | `staticFile(publicPath)` |

导入与 `syncAllAssetsToRemotionPublic` 必须保持上述一致（参见现有 `resolvePublicMirrorPath`）。

### 4.2 拖入时间线（M6-01）

```text
OS 文件 drop 到 TimelineBody
    → importAssetFiles（单批或合并）
    → placeAssetOnTimeline（落点帧 + 轨道）
    → commitTimeline → scheduleGenerate
```

- 与素材库拖入共用 `placeAssetOnTimeline`，不复制第二套落点逻辑
- 桌面直拖与库内拖**最终走同一 place 函数**

### 4.3 片段时长（M6-05）

- 默认：`min(素材.durationInFrames, 成片剩余)`
- 成片不足：弹窗「延长 / 仅放入剩余 / 取消」（剪映式，可配置上限如 30 分钟）
- 延长成片须入撤销栈，并触发 `runGenerate`
- **M5 Agent 不得单独实现另一套延长规则**

### 4.4 删除素材（M6-04）

| 场景 | 行为 |
|------|------|
| 无时间线引用 | 软删除 `isDeleted`，可选删文件 |
| 有引用 | 弹窗：移除相关片段 / 取消 |
| Agent 删除 | 同 UI 规则，经 `main:asset:delete` |

### 4.5 SQLite（M6-08）

- Schema 对齐 [数据库Schema设计.md](数据库Schema设计.md) `assets` 表
- 迁移期可 manifest + DB 双写；**对 M5 暴露的 IPC 返回结构不变**
- SHA-256 去重在 DB 层实现，IPC 仍返回 `ProjectAsset`

### 4.6 UI

- 素材库占 **左栏**（`LeftPanel` / `AssetsPanel`）
- 不占用右侧对话面板区域（M5）
- 批量导入 >50 文件须分批 + 进度（见 [素材处理流水线.md](素材处理流水线.md)）

## 6. IPC 契约（M6 维护）

### 现有

- `main:asset:list`
- `main:asset:import` — `{ filePaths, subprojectPath?, fps? }`
- `main:asset:pickAndImport`

### M6-04 新增（须先更新 [IPC通信协议规范.md](IPC通信协议规范.md)）

- `main:asset:delete` — `{ assetId, mode?: 'soft' | 'removeClips' }`

**返回 `ProjectAsset` 最小集**（M5 依赖，不得破坏性变更）：

`id`, `type`, `name`, `publicPath`, `path`, `durationInFrames?`, `width?`, `height?`

## 7. 验收标准（M6）

- [ ] 桌面 PNG 拖入素材库 → 缩略图 + `assets/images/` + `public/assets/images/`
- [ ] 桌面 MP4 拖入时间线 → 导入 + 片段 + 预览可播
- [ ] 删除被引用素材 → 弹窗 + 可选移除片段
- [ ] 长视频拖入短成片 → 延长弹窗生效
- [ ] 同名文件 → 覆盖/重命名/跳过
- [ ] 预设应用 → 时间线 + 预览正确
- [ ] M5 `importAsset` 联调：仅 IPC 调用即可完整走通
